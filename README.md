# ai-chat-example

This is a simple example of a AI Chat Service using OpenAI API.

## Installation

```bash
# backend
cd hono-app
npm install
```

```bash
# frontend
cd react-app
npm install
```

## Usage

```bash
cd hono-app
# Set your OpenAI API key
export OPENAI_API_KEY=<your key>
# Run the backend
npm run dev
```

```bash
cd react-app
# Run the frontend
npm start
```

Visit http://localhost:3000/
